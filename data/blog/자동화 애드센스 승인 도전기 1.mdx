## 시작하며

부업에 대해 검색하거나 유튜브를 보면 늘 빠지지 않고 등장하는 분야가 있습니다. 바로 블로그로 돈 벌기입니다. 블로그로 수익을 내는 방법은 다양하지만, 그중에서도 가장 잘 알려진 것은 광고 수익을 얻는 방식입니다.

광고 수익을 올리는 방법도 여러 가지가 있지만, 단가가 가장 높은 방법으로 꼽히는 것이 바로 구글 애드센스입니다. 블로그에 애드센스 광고를 달고 승인만 받으면, 콘텐츠를 보는 사람마다 광고 수익이 쌓이기 시작합니다. 하지만 애드센스 승인은 결코 쉽지 않은 관문입니다.

그래서 저는 이번에 한 가지 도전을 해보려고 합니다.  
완전 자동화 애드센스 승인 도전기라는 시리즈를 통해, 인터넷과 유튜브에서 떠도는 애드센스 승인 팁과 규칙을 참고해 완전히 자동화된 방식으로 애드센스 승인을 받아보려고 합니다.

특히 n8n 자동화 툴과 ChatGPT를 활용해, 모든 글을 사람이 직접 작성하지 않고 AI가 작성한 글만으로 승인받는 것을 목표로 합니다. 과연 이런 방식이 실제로 통할 수 있을지 궁금합니다.

이 글에서는 자동화의 모든 과정과 결과를 솔직하게 공유하며, 부업을 꿈꾸는 분들에게 새로운 가능성을 열어드리고자 합니다.

## 애드센스 승인, 유료 강의와 무료 강의에서 배운 핵심 규칙

애드센스 승인을 위해 많은 분들이 다양한 영상과 글을 검색해 보셨을 겁니다. 저 역시 같은 과정을 거쳤습니다. 아래는 제가 유료 및 무료 강의에서 얻은 애드센스 승인에 적합하다고 하는 글 작성 규칙입니다.

#### 애드센스 승인에 적합한 글 작성 규칙

1. **1500자 이상**
2. **HTML 태그를 올바르게 적용한 글**
3. **동일한 분야에서 다양한 내용을 다룬 글**
4. **인터넷에 잘 없는 독창적인 주제를 다룬 글**
5. **이미지 2장 정도 추가**
6. **꾸준한 글 배포**

저는 유료와 무료 강의, 그리고 라이브 방송에서 소개하는 모든 방법을 모아 분석해 보았습니다. 그러나 요즘은 정보의 가치가 가격에 비례하지 않는 것 같습니다. 유료 강의에서 얻는 정보와 인터넷에서 무료로 접할 수 있는 정보는 대부분 비슷하거나 차이가 크지 않습니다.

위와 같은 규칙을 바탕으로, 앞으로 애드센스 승인에 적합한 글을 자동화하여 작성해 나가려고 합니다.

## n8n 워크플로우 설명

n8n을 활용해 애드센스 승인 자동화를 구현하려면 다음 세 가지 설정이 필요합니다.

1. n8n 사용 방법
2. ChatGPT API 사용 방법
3. 워드프레스 API 사용 방법

### n8n 워크플로우 개요

1. ChatGPT를 사용해 미리 정해둔 키워드로 제목을 생성합니다.
2. 1번에서 만든 제목을 기반으로 ChatGPT에서 소주제를 작성합니다.
3. 2번에서 생성한 여러 소주제를 배열 형태로 구성합니다.
   - 배열 형태는 프로그래밍을 통해 쉽게 목록을 조작할 수 있는 형태를 의미합니다.
4. 3번에서 정리한 소주제와 ChatGPT를 활용해 본문 내용을 작성합니다.
5. 최종적으로 워드프레스 블로그에 글을 게시하도록 요청합니다.

이와 같은 흐름으로 n8n 워크플로우가 구성됩니다.  
이 과정에서 ChatGPT는 제목, 소주제, 본문 작성까지 전반적인 글 작성 작업을 담당하며, n8n 워크플로우는 ChatGPT의 어시스턴트 기능을 통해 간단하게 설계할 수 있습니다.
